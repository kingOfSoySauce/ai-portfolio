# 学习笔记结构说明

## 笔记分类

### Daily（每日轻量笔记）
- 文件命名：`daily/day-XX.md`
- 结构：TL;DR + 3 组要点 + TODO
- 用途：快速记录当天学到的内容

### Weekly（周复盘）
- 文件命名：`weekly/week-XX.md`
- 结构：结论 + 依据 + 风险/限制 + 下一步
- 用途：阶段性总结和规划

### Python 基础（语法速查）
- 文件：`python-basics.md`
- 用途：常用 Python 语法和 JS 对照

### 面试卡片（30 秒口述版）
- 文件夹：`interview-cards/`
- 结构：问题 + 一句话结论 + 原理 + 实操 + 坑点 + 项目证据
- 用途：面试准备

---

## 笔记模板

### Daily 模板（轻量版）

```markdown
# Day X - [主题]

## TL;DR（一句话结论）
今天学到的可复述结论

## 3 组要点

### 原理
核心概念是什么

### 实操
我做了什么、怎么验证的

### 坑点
遇到的问题和解决方法

## TODO（下一步）
- [ ] 具体的下一步行动
```

### Weekly 模板（完整版）

```markdown
# Week X - [阶段主题]

## 结论
我完成了什么能力

## 依据
- 做了哪些改动
- 对比实验结果
- 关键指标

## 风险/限制
还不行的点、已知问题

## 下一步（MECE 拆解）
1. 
2. 
3. 
```

---

## 使用建议

1. **平时学习**：只写 Daily（轻量），不要强迫自己写完整版
2. **周末复盘**：整理成 Weekly，梳理进度和计划
3. **遇到问题**：用 MECE 拆分排查（记录在 Daily 的坑点部分）
4. **准备面试**：把 Daily 压缩成面试卡片
